# TOKEN LISTING POLICY

## Eligibility Criteria

Before any token is listed on TROPTIONS platforms, the following must be satisfied:

### 1. Legal Structure
- [ ] Entity formed in compliant jurisdiction
- [ ] Operating Agreement / Articles filed
- [ ] Registered agent active
- [ ] Good standing confirmed

### 2. Securities Compliance
- [ ] Counsel opinion: not a security OR
- [ ] Reg D/Reg S exemption filed OR
- [ ] Full registration statement effective
- [ ] No active SEC/state enforcement

### 3. Technical Verification
- [ ] Smart contract audit complete
- [ ] Mint authority verified/renounced
- [ ] Treasury/multisig disclosed
- [ ] Token address confirmed on-chain

### 4. Financial Requirements
- [ ] Legal defense escrow funded ($25K minimum)
- [ ] Initial liquidity committed
- [ ] Fee structure documented
- [ ] Revenue share mechanism defined

### 5. Operational Readiness
- [ ] KYC/AML policy implemented
- [ ] Exchange listings confirmed (if applicable)
- [ ] Marketing plan approved
- [ ] Support channels established

## Delisting Criteria

Tokens may be delisted for:
- Security law violation
- Regulatory enforcement
- Material misrepresentation
- Technical failure/exploit
- Failure to maintain escrow
