## PURPOSE

This matrix defines where TROPTIONS, INC. (Wyoming) and the Exchange OS may lawfully on-board users, list tokens, and facilitate real estate-linked transactions. It is a **living document** that feeds directly into the Exchange OS IP geolocation middleware.

**Legal Principle:** TROPTIONS is a non-custodial trade-currency infrastructure. Where a jurisdiction requires custodial licensing, securities registration, or money-transmitter status that TROPTIONS does not hold, that jurisdiction is **Blocked** or **Restricted** until compliance is attained.

## EXCHANGE OS IMPLEMENTATION

### Geo-Blocking Middleware (Pseudocode)
```javascript
// Express.js / Cloudflare Worker middleware
const BLOCKED_STATES = ['CA', 'NY', 'TX', 'HI'];
const RESTRICTED_STATES = ['AL', 'AR', 'CT', 'IL', 'LA', 'MD', 'MA', 'MN', 'MO', 'NV', 'NC', 'RI', 'SC', 'WA'];

function jurisdictionGate(req, res, next) {
  const region = req.geo?.region || geoLocate(req.ip);

  if (BLOCKED_STATES.includes(region)) {
    return res.status(403).json({
      error: 'Jurisdiction Prohibited',
      message: 'TROPTIONS Exchange OS is not available in your jurisdiction pending compliance review.',
      code: 'JURISDICTION_BLOCKED'
    });
  }

  if (RESTRICTED_STATES.includes(region)) {
    if (!req.user?.accreditedInvestor && !req.user?.micaAcknowledged) {
      return res.status(403).json({
        error: 'Accredited Investor Required',
        message: 'Your jurisdiction requires accredited investor status or regulatory acknowledgment.',
        code: 'ACCREDITED_REQUIRED'
      });
    }
  }

  next();
}
```

### Wallet Attestation Requirement
For RESTRICTED jurisdictions, users must sign a message with their wallet:
```
"I certify that I am acquiring [TOKEN] for consumptive utility purposes only,
not as an investment, and I am an accredited investor under Rule 501 of
Regulation D (if required by my jurisdiction)."
```
