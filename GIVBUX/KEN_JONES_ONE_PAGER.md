## How engagement starts

1. **NDA** and written discovery (users, budget, regulatory status, merchant data)  
2. **Escrow** for initial legal/architecture phase  
3. **Signed license** — TROPTIONS retains IP; licensee receives revocable API access  
4. **Phased delivery** — testnet before production keys  

No production access, no repository access, and no custom engineering until escrow and structure are agreed.
